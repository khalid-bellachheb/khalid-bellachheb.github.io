My Projects

English Proficiency Prediction NLP

June 2021 - August 2021

Associated with: École Nationale d'Ingénieurs de Brest (ENIB)

Basically the idea of the project is to predict someone's English proficiency based on a text input.

We used the NICT JLE Corpus, which consists of transcripts from 1,281 participants' English oral proficiency interviews. Each participant received a Standard Speaking Test (SST) score between 1 (low proficiency) and 9 (high proficiency).

  • Pre-process the dataset: extract the participant transcript and remove unnecessary tags
  • Process the dataset: extract features using the Bag of Words (BoW) technique
  • Train a classifier to predict the SST score
  • Compute the accuracy and plot the confusion matrix
  • Try to improve the system using GloVe instead of BoW

Skills: Natural Language Processing, NLTK, Tokenization, Lemmatization, Bag of Words, Vector Embedding, Keras, Neural Networks, Confusion Matrix
